# 🔥 AshQit – Premium Security Toolkit

![Version](https://img.shields.io/badge/version-1.0-blue)
![Platform](https://img.shields.io/badge/platform-Linux-lightgrey)

> **A modular, terminal‑based security arsenal built for ethical hackers, penetration testers, and security enthusiasts.**

AshQit is a **personal project** that brings together essential security tools into a single, beautifully crafted command‑line interface. It combines Python and Bash to deliver a premium experience for reconnaissance, web testing, network analysis, cryptography, forensics, and utilities – all under one roof.

## ✨ Features

- **Beautiful Terminal UI** – Colored output, ASCII art, and a clean menu system.
- **8 Essential Tool Categories** – Covers the most common security tasks.
- **Modular Architecture** – Easy to extend with new tools.
- **Lightweight & Fast** – Minimal dependencies, built for performance.
- **Educational Focus** – Each tool is designed to teach core security concepts.

## 🛠️ Tools Included (v1)

### 🔍 RECONNAISSANCE
| Tool | Description |
|------|-------------|
| `subfinder` | Enumerate subdomains using dictionary brute‑force with concurrent requests. |

### 🌐 WEB TESTING
| Tool | Description |
|------|-------------|
| `dirbust` | Discover hidden directories and files on web servers. Handles custom extensions and recursion. |

### 🌍 NETWORK
| Tool | Description |
|------|-------------|
| `arp` | ARP spoofing tool for man‑in‑the‑middle attacks (educational / lab use only). |

### 🔐 CRYPTOGRAPHY
| Tool | Description |
|------|-------------|
| `sslcheck` | Retrieve and analyse SSL/TLS certificates: issuer, expiry, weak protocols, and more. |

### 🔑 PASSWORD
| Tool | Description |
|------|-------------|
| `genpass` | Generate strong passwords with custom length, character sets, and entropy scoring. |

### 🔬 FORENSICS
| Tool | Description |
|------|-------------|
| `analyze` | Extract file metadata, magic bytes, hash values (MD5/SHA1/SHA256), and entropy. |

### 🛠️ UTILITIES
| Tool | Description |
|------|-------------|
| `macchanger` | Change or randomize network interface MAC addresses. |
| `iptools` | IP address validation, CIDR range calculation, integer conversion, and reverse DNS. |
