# Browser Data Exposure

An educational cybersecurity project demonstrating how browser-stored data can be accessed in insecure environments, paired with a simple game interface to illustrate social engineering concepts.

---

## Overview

This project highlights how seemingly harmless applications can be used to demonstrate risks related to **data exposure and user trust**. It is designed strictly for studying security vulnerabilities and raising awareness about safe computing practices.

---

## Features

- 🎮 Simple game interface (user-facing component)  
- 🔍 Demonstration of browser data access patterns  
- 📄 Generates a structured extraction report (`browser_data.txt`)  
- 🧪 Designed for controlled, local testing environments
