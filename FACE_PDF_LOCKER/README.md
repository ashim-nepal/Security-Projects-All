# Face‑Secure PDF Locker

An educational cybersecurity project demonstrating how **face recognition** can be combined with **strong encryption** to protect sensitive PDF files. It highlights the importance of biometric authentication and secure file handling.

---

## Overview

This project showcases a desktop application that uses a live webcam to register a user’s face, then encrypts a PDF with AES‑256. The file can only be decrypted after the same face is verified again. It is designed for studying real‑world security concepts in a controlled, local environment.

---

## Features

- 🔐 **AES‑256 encryption** of PDF files  
- 🧠 **Face recognition** using dlib and the `face_recognition` library  
- 🎥 **Live webcam capture** for registration and verification  
- 🖥️ **Simple Tkinter GUI** for easy interaction  
- 🗂️ **Secure storage** – the original PDF is removed after encryption

## Disclaimer

- This project is strictly for **educational and ethical cybersecurity research**  
- It demonstrates potential security mechanisms to encourage **better protection practices**  
- Do not use on real sensitive documents without understanding the risks (keys are stored locally)

---

## Learning Focus

- Understanding biometric authentication (face recognition)  
- Implementing symmetric encryption (AES‑CBC)  
- Building a graphical user interface with Tkinter  
- Exploring secure file handling and key management
