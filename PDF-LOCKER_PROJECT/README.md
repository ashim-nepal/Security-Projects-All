# PDF Locker (pikepdf + Tkinter)

A simple desktop utility for securing PDF files with password protection using a clean graphical interface.

---

## Overview

This project provides an easy way to lock PDF files by applying password encryption. Built with `pikepdf` for reliable PDF handling and `tkinter` for a lightweight GUI, it allows users to secure documents without needing complex tools.

---

## Features

- 🔐 Add password protection to PDF files  
- 🖥️ Simple and user-friendly interface (Tkinter)  
- ⚡ Fast processing with minimal dependencies  
- 📂 Save protected files as new encrypted PDFs  

---

## How It Works

1. Select a PDF file  
2. Enter a password  
3. The tool locks the file  
4. A new locked PDF is saved to disk  

---

## Output

- `locked_<filename>.pdf` — PDF file
